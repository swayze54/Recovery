# Recovery

## Description
- Program that is able to recover .jpeg deleted images from memory cards initialized with FAT system

## How to Run 
- Copy program into a text editor with Bash terminal (or prefered terminal) and Clang compiler. Copy the .raw image file to your working directory. 

## Usage 
- In the command line, complile the recovery file into an executable. whem executing, you will be required to have two command line arguements, the second one being your imported image files name. After execution, you will the recovered imaged in your working directory.
