# Recovery

## Description
- Program that will recover .jpeg deleted images from memory cards initialized with FAT system

## How to Run 
- Copy program into a text editor with Bash terminal (or preferred terminal) and Clang compiler. Copy the .raw image file to your working directory. 

## Usage 
- In the command line, compile the recovery file into an executable. When executing, you will be required to have two command line arguments, the second one being your imported image files name. After execution, you will have the recovered images in your working directory.
